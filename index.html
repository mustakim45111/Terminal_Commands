CMD, PowerShell ও Git Bash এর গুরুত্বপূর্ণ কমান্ডসমূহ

১. CMD (Command Prompt) কমান্ডসমূহ

কমান্ড ব্যাখ্যা (বাংলায় সহজ)
dirবর্তমান ফোল্ডারে থাকা ফাইল ও ফোল্ডারের তালিকা দেখায়।
cdফোল্ডার পরিবর্তন করে। যেমন cd Documents মানে Documents ফোল্ডারে যাওয়া।
mdনতুন ফোল্ডার তৈরি করে। যেমন md newfolder নতুন ফোল্ডার তৈরি করবে।
rdফোল্ডার মুছে ফেলে। যেমন rd oldfolder।
delফাইল মুছে ফেলার জন্য। যেমন del file.txt।
copyফাইল কপি করে। যেমন copy file.txt d:\backup
moveফাইল বা ফোল্ডার অন্য জায়গায় সরায়।
clsকমান্ড প্রম্পট পর্দা পরিষ্কার করে।
ipconfigকম্পিউটারের আইপি ঠিকানা ও নেটওয়ার্ক তথ্য দেখায়।
pingইন্টারনেটে কোনো ঠিকানায় পিং পাঠিয়ে তা চালু আছে কিনা দেখে।
exitCMD বন্ধ করে।

২. PowerShell কমান্ডসমূহ

কমান্ড ব্যাখ্যা (বাংলায় সহজ)
Get-ChildItemবর্তমান ফোল্ডারের ফাইল ও ফোল্ডার দেখায় (CMD এর dir এর মত)।
Set-Locationফোল্ডার পরিবর্তন করে (CMD এর cd এর মত)।
New-Item -ItemType Directory -Name "foldername"নতুন ফোল্ডার তৈরি করে।
Remove-Itemফাইল বা ফোল্ডার মুছে ফেলে।
Copy-Itemফাইল বা ফোল্ডার কপি করে।
Move-Itemফাইল বা ফোল্ডার সরায়।
Clear-Hostপাওয়ারশেল পর্দা পরিষ্কার করে (CMD এর cls)।
Get-Processবর্তমানে চলমান প্রক্রিয়া বা প্রোগ্রামগুলো দেখায়।
Stop-Process -Name processnameনির্দিষ্ট প্রক্রিয়া বন্ধ করে।
Get-Helpকোনো কমান্ডের সাহায্যের তথ্য দেখায়।
ExitPowerShell বন্ধ করে।

৩. Git Bash কমান্ডসমূহ

কমান্ড ব্যাখ্যা (বাংলায় সহজ)
lsবর্তমান ফোল্ডারের ফাইল ও ফোল্ডারের তালিকা দেখায় (CMD এর dir এর মত)।
cdফোল্ডার পরিবর্তন করে।
mkdirনতুন ফোল্ডার তৈরি করে।
rmফাইল বা ফোল্ডার মুছে ফেলে।
cpফাইল বা ফোল্ডার কপি করে।
mvফাইল বা ফোল্ডার সরায়।
clearটার্মিনাল পর্দা পরিষ্কার করে।
pwdবর্তমান ডিরেক্টরির সম্পূর্ণ পথ দেখায়।
git statusগিট রিপোজিটরির বর্তমান অবস্থা দেখায়।
git add .সকল পরিবর্তিত ফাইল স্টেজ করে commit এর জন্য।
git commit -m "message"পরিবর্তনগুলো একটি বার্তা সহ সংরক্ষণ করে।
git pushলোকাল রিপোজিটরির পরিবর্তনগুলো রিমোটে পাঠায়।
git pullরিমোট রিপোজিটরি থেকে নতুন পরিবর্তন নিয়ে আসে।
exitটার্মিনাল বন্ধ করে।
